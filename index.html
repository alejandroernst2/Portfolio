CONCEPTOS BÁSICOS PARA EL DESARROLLO DE TU WEB

¿QUE ES UNA PÁGINA WEB?

Una pagina web es uno de los archivos que conforma el sitioweb, como por ejemplo, la página de inicio, de contacto, de experiencias de clientes, o de trabajos realizados por el propietario del sitio.

¿QUE ES UN SITIO WEB?

Un sitio web es un conjunto de archivos que incluye diferentes páginas web.

¿QUÉ ES UNA SECCIÓN?

Una sección es un fragmento marcado de la pagina web.

¿CUANTO DEMORA EL DESARROLLO DE UNA PAGINA WEB?

Dependiendo del proyecto que deseas armar, pero el plan básico tarda aproximadamente entre 2 y 3 meses.
